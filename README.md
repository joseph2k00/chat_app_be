# Chat App Backend
API-based backend for Chat App with OpenAI API integration for text translation with Web Socket and JWT authentication

# Tech Stack
- **Language:**
> PHP (Laravel)
- **Libraries:**
> Laravel Reverb (Web Socket) - https://github.com/laravel/reverb
> tymon/jwt-auth - https://github.com/tymondesigns/jwt-auth
- **Third-Party APIs:**
> OpenAI API
